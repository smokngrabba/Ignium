##  Configuration

Ignium utilizes a streamlined **`tables.py`** system for rapid, automated builds.

1.  **Open** `tables.py` in your preferred editor.
2.  **Define** your target Discord Webhook URL.
3.  **Toggle** your build parameters:
    *   `OBFUSCATION`: Enable code logic protection.
    *   `JUNK_CODE`: Apply file size padding for evasion.
    *   `FAKE_ERROR`: Trigger a custom system alert on execution.
4.  **Brand** your build with a custom `.ico` file path.

Generate your custom payload with a single command:
